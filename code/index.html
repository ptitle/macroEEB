Code

R packages that I maintain and contribute to:

BAMMtools: Provides tools to process and interrogate analyses conducted with BAMM.

envirem: Functions to generate the ENVIREM set of climatic rasters.

rangeBuilder: Miscellaneous tools for cleaning species occurrence records and generating geographic range polygons via alpha hulls.

EcoPhyloMapper (epm): Platform for integrating geographic range, phylogeny and morphometric data.

Miscellaneous code

bammCheck.R: R function to generate a plot containing information for diagnosing the convergence of a BAMM analysis.
